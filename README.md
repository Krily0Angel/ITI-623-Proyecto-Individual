# ITI-623-Proyecto-Individual
# Sistema de Inventario de Productos

# Descripción del proyecto
Este proyecto es una aplicación web para la gestión de productos en un inventario.  
Permite registrar, visualizar, actualizar y eliminar productos, además de realizar búsquedas por nombre o categoría.  
La aplicación está pensada para pequeñas tiendas, bodegas o uso personal en el hogar.  
Cada producto puede incluir una imagen, nombre, categoría, cantidad disponible y precio.

---

# Funcionalidades principales
- Crear: agregar nuevos productos al inventario con su información y una imagen.  
- Leer: visualizar la lista completa de productos registrados.  
- Actualizar: editar datos de un producto existente (precio, cantidad, nombre, etc.).  
- Eliminar: eliminar productos que ya no están disponibles.  
- Buscar: filtrar productos por nombre o categoría.  
- Listas personalizadas: posibilidad de mostrar productos con bajo stock o por tipo de artículo.
